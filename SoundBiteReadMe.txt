SoundBite Readme



Overview

The purpose of this application is to show and answer the developing of a custom sound board for entertainment and application use. The application will be able to record, manipulate and play back audio files as well as create custom samples that will allow you to alter pitch, modulation, and volume. 
 



Hardware and Software
 
This is coded for Android phones running on OS ice-cream 4.0 sandwich or any later version. These versions include – Jelly Bean, KitKat, Lollipop, Marshmallow, Nought, and Oreo. The external peripherals used are an Android phones microphone and speakers. 
 
 
 
Run Instructions
 
To run the completed app, download SoundBite off the Google Play store. Launch the app which will put you on the main screen.
